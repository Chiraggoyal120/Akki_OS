# Browser Automation Skill

## Purpose
LinkedIn aur Twitter pe browser se automatically post karna

## Setup (One Time)
Fresh login karna padega:
node setup-linkedin.js
node setup-twitter.js

## Usage
# LinkedIn post karo
node linkedin-actions.js

# Twitter post karo
node twitter-actions.js

## Requirements
- Playwright installed
- Browser sessions saved
- npx playwright install chromium

## Note
Browser sessions machine specific hoti hain
Har naye VPS pe fresh login karna padega
