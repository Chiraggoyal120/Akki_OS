# Workspace Skill

## Purpose
Main agent ka brain - identity, instructions aur memory

## Files
- SOUL.md: Main agent ki identity aur personality
- INSTRUCTIONS.md: Detailed workflow instructions

## How it works
1. Agent starts → SOUL.md padhta hai
2. Task milta hai → INSTRUCTIONS.md follow karta hai
3. Skills use karta hai → Kaam karta hai
4. Supabase mein save karta hai
5. Webhook report karta hai

## Customize karo
SOUL.md mein apni details daalo:
- Your name
- Your business
- Your tone
- Your goals
